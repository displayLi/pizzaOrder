## Build Setup

``` bash
# install dependencies
$ npm install
$ npm install vuex
$ npm install vue-router
$ npm install axios

# serve with hot reload at localhost:8080
$ npm run dev

# build for production with minification
$ npm run build

#json-server install
$ cd pizzaOrder/local
$ npm install -g json-server 
$ npm install json-server
$ json-server --watch db.json
